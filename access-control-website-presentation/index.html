Sobre Nós

Quem Somos

Somos um grupo de estudantes da Escola Profissional Bento de Jesus Caraça (EPBJC), matriculados no 12º ano do curso de Gestão e Programação de Sistemas Informáticos (GPSI) na turma A. Temos o prazer de apresentar o nosso projeto inovador denominado "Controlo de Acesso".

Nossas Competências Técnicas

conhecimentos básicos em Hardware, Arduíno, CSS, HTML e JS.

Projeto

Projeto controlo de Acesso

O objetivo do nosso projeto é criar um sistema seguro, efetivo e inovador de um controlo de acesso, utilizando um leitor de CD e um padrão de cordas de guitarra para acionar a abertura e o fecho de uma porta. O projeto também envolve o uso do Arduíno para programar o código necessário para executar essas.
